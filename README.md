# 🥬 韭菜后花园 — A股低估值筛选系统

> 别在涨停板上追高了，这里帮你找那些被市场遗忘、跌到地板价还要往下挖地窖的白马股。

## 这是什么

一套基于腾讯+新浪实时数据的筛选引擎，每天自动扫描 A 股，找出同时满足三个条件的标的：

1. **便宜** — PB 破净、PE 个位数，价值分拉满
2. **惨** — 距历史高点腰斩起步，跌幅越大越兴奋
3. **躺平** — 股价在底部几乎不动，均线粘得像胶水，随时可能变盘

综合评分 Top N 输出 JSON + Markdown 报告，附带 Core/Radar 双池淘汰制，不让垃圾股赖在池子里混日子。

## 一分钟跑起来

```bash
# 日常筛选 Top 10（默认 12 个月周期）
python3 stock_screen.py

# 拉长到 5 年找更深的地窖
python3 stock_screen.py --months 60

# 自定义数量，不写历史
python3 stock_screen.py --top 20 --no-history

# 每周全市场大扫除（淘汰+补入+晋升）
python3 stock_screen.py --maintain
```

| 参数 | 默认 | 说明 |
|------|------|------|
| `--top N` | 10 | 输出 Top N |
| `--months N` | 12 | 评测周期 1~120 月 | 建议: 短线 1~6 / 中长线 12~36 / 超长线 60~120 |
| `--no-history` | — | 跳过 history.json 写入 |
| `--maintain` | — | 全市场扫描 + 淘汰 + 补入 + 晋升 |

## 免责声明

这不是投资建议。筛选出来的股票只是数学上符合"超跌横盘"形态，不代表明天就会涨。可能继续横一年，可能再跌 30%，也可能退市。韭菜也是菜，投资需谨慎。
